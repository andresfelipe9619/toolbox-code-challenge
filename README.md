# Toolbox: Full Stack JS - Code Challenge

## Commands

- Install API dependencies

```cmd
cd ./api/
npm install
```

- Run API tests

```cmd
cd ./api/
npm run test
```

- Run API app

```cmd
cd ./api/
npm start
```

- Install client dependencies

```cmd
cd ./client/
npm install
```

- Run client app

```cmd
cd ./client/
./client/npm start
```

## Running with Docker

- Run containers

```cmd
docker-compose up
```

- Stop and remove containers

```cmd
docker-compose down
```
