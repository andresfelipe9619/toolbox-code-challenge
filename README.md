# Toolbox: Full Stack JS - Code Challenge

## Commands

- Run api app

```cmd
./api/npm start
```

- Install client dependencies

```cmd
./client/npm install
```

- Run client app

```cmd
./client/npm start
```

- Run cypress tests

```cmd
cd ./cypress
npm run cy:run
```


## Running with Docker

- Run containers

```cmd
docker-compose up
```

- Stop and remove containers

```cmd
docker-compose down
```
